# Início Rápido

O **Avalia** foi projetada para tornar o processo de avaliação o mais simples e eficiente possível, permitindo que você, como avaliador, se concentre no que realmente importa: analisar e avaliar os projetos. Aqui está um guia rápido para começar:

### Acesso ao Sistema

O login para os avaliadores é feito de maneira prática e direta, utilizando um **link de acesso personalizado**. Esse link é enviado para você via e-mail ou WhatsApp, dependendo das informações fornecidas à administração da feira durante seu cadastro. O link se parecerá com o seguinte exemplo:

```
https://avalia.hotay.dev/evaluator?code=xxxx-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

#### Acesso pelo Celular ou Computador

Você pode **usar seu próprio celular** para acessar o sistema através do link enviado. Se preferir, a administração da feira também disponibiliza computadores para a avaliação. Nesse caso, você pode solicitar ajuda à administração para realizar o login. Esse login é feito através de um **código único** que você receberá da administração. Para acessar manualmente, utilize o seguinte link:

- [https://avalia.hotay.dev/evaluator](https://avalia.hotay.dev/evaluator)

### Navegação na Plataforma

Assim que acessar o sistema, você será direcionado automaticamente à **lista de projetos** que foram atribuídos a você para avaliação. A administração da feira cuida da distribuição dos projetos, levando em consideração sua área de atuação e formação profissional, garantindo que você avalie os projetos mais adequados ao seu perfil.

### Avaliando os Projetos

Para avaliar um projeto, basta clicar no nome do projeto na lista. Isso o levará diretamente à **tela de avaliação**, onde você encontrará perguntas com **notas quantitativas** (de 5 a 10) que devem ser respondidas. Há também um campo opcional para **comentários adicionais**, caso deseje fornecer uma avaliação mais detalhada.

Após preencher as notas e, opcionalmente, os comentários, basta clicar em **Enviar**. Você será redirecionado de volta à lista de projetos, onde poderá ver claramente quais projetos já foram avaliados e quais ainda precisam ser avaliados.

### Dicas Finais

- **Verifique seu e-mail ou WhatsApp** para encontrar o link de acesso.
- Se precisar de ajuda ou preferir não usar seu dispositivo pessoal, solicite o apoio da administração da feira.
- Lembre-se de revisar suas avaliações antes de enviá-las para garantir que todos os critérios foram corretamente preenchidos.

Com esses passos simples, você estará pronto para começar a avaliação dos projetos, contribuindo para o sucesso da feira científica e para o desenvolvimento dos alunos.
