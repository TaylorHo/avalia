# Usuários Administrativos

No **Avalia**, é possível convidar outros usuários para participar da administração da feira de forma simples e eficaz. Aqui estão os passos e informações importantes sobre o processo de convite:

### Convidando Usuários via E-mail

Como administrador, você pode convidar outros usuários para se juntarem à administração da feira **usando o e-mail** deles. Ao convidar alguém, o sistema registra o convite, mas **não envia automaticamente nenhum tipo de notificação ou link para o usuário convidado**. A comunicação do convite é feita manualmente.

### Como Enviar o Link de Convite

Após convidar um novo usuário, você verá uma opção na página de **Usuários Administrativos** chamada **"Copiar Link de Convite"**. Esse link é personalizado e permite que o usuário convidado se junte à administração da feira. 

Você pode copiar esse link e enviá-lo diretamente ao usuário via e-mail, WhatsApp ou qualquer outro meio de comunicação. Quando o usuário clicar no link, ele será direcionado para a página de cadastro, onde poderá se cadastrar usando uma conta Google ou via link enviado por e-mail. 

### Acesso Imediato à Feira

Depois de se cadastrar através do link de convite, o novo usuário terá acesso imediato à feira, sem a necessidade de passar por qualquer configuração inicial. Isso facilita a entrada de novos administradores, que já poderão começar a colaborar na gestão da feira assim que completarem o cadastro.

### Acesso Sem o Link de Convite

Mesmo se o usuário convidado não receber ou não usar o link de convite, ele ainda pode se juntar à feira. Se o usuário se cadastrar utilizando o **mesmo e-mail que foi convidado**, ele terá acesso imediato à feira após o login, sem passar por etapas de configuração. O sistema reconhece o e-mail e automaticamente atribui o usuário à administração da feira.

### Resumo dos Passos

1. Convide um usuário administrativo via e-mail.
2. Copie o link de convite e envie-o diretamente ao usuário convidado.
3. O usuário se cadastra através do link com o e-mail convidado.
4. O usuário terá acesso imediato à feira sem precisar configurar nada adicional.

Com esses passos simples, você pode expandir a equipe de administração da sua feira de forma rápida e sem complicações.
