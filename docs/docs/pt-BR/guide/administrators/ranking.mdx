# Classificações / Ranking

A página de **Ranking/Classificação de Projetos** do **Avalia** permite que os administradores acompanhem a classificação dos projetos avaliados, organizados por categoria. A seguir, detalhamos as funcionalidades dessa página e como utilizá-las:

### Organização por Categorias

A página de Ranking é a **página inicial** da área de administração, exibindo os projetos classificados com base nas categorias previamente definidas na **planilha de fonte de dados**.

Para cada categoria, a página mostra um **bloco** que contém os 5 primeiros projetos com a maior pontuação. Isso facilita a visualização rápida dos projetos de destaque em cada área.

### Visualização Completa de Classificações

Ao lado de cada bloco de categoria, há um botão de **lupa**. Clicando nele, você pode visualizar a **lista completa de projetos** avaliados dentro daquela categoria. Os projetos são exibidos em ordem de classificação, da maior para a menor pontuação. 

Isso permite que você veja não apenas os melhores colocados, mas também o desempenho de todos os projetos participantes da categoria.

:::info
A nota mostrada na lista é uma média das notas obtidas na feira. A nota varia de 5 a 10 e foi pensada desta forma para facilitar a atribuição de notas baseando-se no sistema de notas usado no Ensino Médio Brasileiro. Em breve iremos adicionar outras opção, como notas de 1 a 5 e de 1 a 10.
:::

### Atualização do Ranking

A atualização da lista de classificações não ocorre de forma automática após o primeiro carregamento da página. 

Na primeira vez que a página é acessada, as informações são carregadas automaticamente, mas após isso, é exibida uma mensagem indicando **quando foi feita a última atualização** dos dados de classificação.

Para atualizar manualmente as informações, você pode clicar no botão de **"atualizar"** localizado ao lado da mensagem da data de última atualização. Essa funcionalidade é útil para garantir que você esteja visualizando as classificações mais recentes, especialmente se novos projetos foram avaliados desde a última visualização.

### Resumo das Funcionalidades

1. **Exibir os 5 principais projetos** em cada categoria, conforme definido na planilha de dados.
2. **Visualizar a lista completa de projetos** em ordem de classificação dentro de cada categoria.
3. **Atualizar manualmente** a lista de classificações para garantir que os dados exibidos sejam os mais recentes.
