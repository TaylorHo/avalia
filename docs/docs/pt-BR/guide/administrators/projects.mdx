# Projetos

A página de **Projetos** do **Avalia** permite que os administradores gerenciem os projetos inscritos na feira. A seguir, detalhamos as principais funcionalidades e como utilizá-las:

### Navegando na Página de Projetos

Na página de Projetos, você encontrará uma **tabela** que exibe informações essenciais sobre cada projeto inscrito, incluindo:

- **Título do Projeto**: O nome ou título oficial do projeto.
- **ID**: Um identificador único atribuído ao projeto.
- **Área do Projeto**: O campo ou categoria à qual o projeto pertence.
- **Quantidade de Avaliadores**: O número de avaliadores designados para avaliar o projeto.
- **Avaliações Concluídas**: Quantas das avaliações já foram realizadas.

### Atualização da Lista de Projetos

Semelhante à página de Avaliadores, a lista de projetos é atualizada automaticamente **somente na primeira vez** que a página é carregada. Após essa atualização inicial, uma mensagem indicará **há quanto tempo** a lista foi atualizada.

Para **atualizar manualmente** a lista de projetos, clique no ícone de **"atualizar"** localizado no canto superior direito da página. Isso é útil caso novos projetos sejam adicionados por outros administradores enquanto você está navegando na página.

### Adicionando um Novo Projeto

Ao lado do botão de atualizar, há o botão **"Adicionar Novo Projeto"**. Clicando nele, você pode incluir um novo projeto fornecendo as seguintes informações:

- **Título**: O nome ou título do projeto.
- **Descrição**: Um breve resumo ou explicação sobre o projeto.
- **Área do Projeto**: A categoria ou área temática à qual o projeto pertence.

Após a inserção dessas informações, o novo projeto será adicionado **automaticamente** à lista que você está visualizando. No entanto, a lista completa de projetos (incluindo aqueles adicionados por outros administradores) **não será atualizada automaticamente**. Para visualizar todos os novos projetos, inclusive os adicionados por outros usuários administrativos, você precisará clicar novamente no botão de atualizar. Ao lado do botão há uma mensagem mostrando há quanto tempo foi realizada a última atualização.

:::info
Novos projetos também podem ser adicionados diretamente na planilha de fonte de dados, tomando apenas o cuidado de não editar as células que contenham o aviso de "Não editar".
:::

### Atribuição de Avaliadores

Diferente da página de Avaliadores, a **atribuição de avaliadores a um projeto** não é feita diretamente nesta página. A designação de quais avaliadores avaliarão um projeto específico é gerenciada na página de Avaliadores, onde você pode atribuir projetos aos avaliadores conforme a área de atuação e disponibilidade.

### Resumo das Funcionalidades

1. **Visualizar** e **gerenciar** a lista de projetos inscritos.
2. **Atualizar a lista** de projetos manualmente.
3. **Adicionar novos projetos** com título, descrição e área.
4. **Acompanhar** o progresso das avaliações atribuídas a cada projeto.
