# Uso comercial (instituições privadas)

## Gratuidade para Escolas Públicas

**O Avalia** é um sistema totalmente gratuito para **escolas públicas** e **instituições sem fins lucrativos**. Esse modelo de gratuidade reflete nosso compromisso com a educação pública, garantindo que todas as escolas tenham acesso a uma ferramenta eficaz para avaliação de feiras científicas sem nenhum custo.

## Licenciamento para Escolas Particulares e Instituições Privadas

Embora o sistema seja gratuito para escolas públicas, **escolas particulares** e **feiras de instituições privadas** podem utilizar o Avalia sob um modelo de licenciamento diferenciado. Nessas situações, o uso do sistema não é gratuito.

### Personalização e Licenciamento

Com o licenciamento pago, é removida a mensagem padrão de "**Desenvolvido gratuitamente pela Hotay**". Em seu lugar, é possível adicionar o **logo** da escola ou da feira, personalizando ainda mais a experiência para a instituição.

### Valor do Licenciamento

O custo do licenciamento é de **R$200/ano**, permitindo que a escola ou instituição privada utilize o sistema **quantas vezes forem necessárias** durante o ano. É importante notar que este licenciamento é exclusivo para uma única escola ou instituição e **não pode ser compartilhado** com outras.

Para solicitar acesso à versão comercial do sistema, [entre em contato com a Hotay clicando aqui](mailto:contato@hotay.dev).

## Compromisso com a Gratuidade para Escolas Públicas

Reiteramos que o **Avalia** é e sempre será gratuito para **escolas públicas** e **instituições sem fins lucrativos**. Nosso objetivo é proporcionar acesso igualitário a ferramentas tecnológicas, garantindo que as escolas públicas possam continuar utilizando o sistema sem qualquer custo.
